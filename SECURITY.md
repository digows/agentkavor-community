# Security Policy

## Supported versions

Security fixes target the latest stable Kavor release. Pre-release builds receive best-effort support.

## Report a vulnerability

Do not open a public Issue or Discussion for a suspected vulnerability.

Use [GitHub Private Vulnerability Reporting](https://github.com/digows/agentkavor-community/security/advisories/new). If that channel is unavailable, email [contact@agentkavor.com](mailto:contact@agentkavor.com).

Include the affected version, impact, reproduction steps, and any suggested remediation. Do not access data that is not yours, disrupt services, or publicly disclose the issue before a fix can be coordinated.
