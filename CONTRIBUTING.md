# Contributing

Thank you for helping improve Kavor.

## Choose the right channel

- Use **Discussions / Ideas** for product ideas and feature requests.
- Use **Discussions / Q&A** for questions and usage guidance.
- Use **Issues** only for reproducible bugs.
- Use **Private Vulnerability Reporting** for security concerns. Never disclose a vulnerability publicly.

Search before posting. If an existing Discussion covers your idea, upvote it and add any new context there instead of creating a duplicate.

## Public information

Everything posted in Discussions and Issues is public. Remove credentials, source code, terminal output, private repository details, personal data, and confidential Workspace information before submitting.

## Pull requests

This repository contains community configuration only. Pull requests may improve its forms or community documentation. Product source changes are not accepted here.

By participating, you agree to follow the [Code of Conduct](CODE_OF_CONDUCT.md).
