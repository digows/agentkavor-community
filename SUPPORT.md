# Kavor Support

## Questions and workflows

Use [Kavor Q&A](https://github.com/digows/agentkavor-community/discussions/categories/q-a) for public questions about
installation, product behavior, and coding-agent workflows.

Search existing Discussions first. Include your Kavor version and operating system when they affect the answer.

## Ideas

Use [Kavor Ideas](https://github.com/digows/agentkavor-community/discussions/categories/ideas) to describe a missing
workflow or product opportunity. Focus on the problem and desired outcome; a complete design is not required.

## Bugs

Use the [bug report form](https://github.com/digows/agentkavor-community/issues/new?template=bug-report.yml) for
reproducible product defects.

The latest stable Kavor release receives active support. Pre-release builds receive best-effort support and may
contain incomplete behavior.

## Security and privacy

Never publish credentials, private source code, personal data, confidential Workspace information, or a suspected
vulnerability.

Report security concerns through
[GitHub Private Vulnerability Reporting](https://github.com/digows/agentkavor-community/security/advisories/new).
See [SECURITY.md](SECURITY.md) for the complete policy.
