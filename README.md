## Participate

- [Share an idea](https://github.com/digows/agentkavor-community/discussions/categories/ideas)
- [Browse and vote on ideas](https://github.com/digows/agentkavor-community/discussions/categories/ideas)
- [Ask the community](https://github.com/digows/agentkavor-community/discussions/categories/q-a)
- [Report a bug](https://github.com/digows/agentkavor-community/issues/new?template=bug-report.yml)
- [Report a security vulnerability privately](https://github.com/digows/agentkavor-community/security/advisories/new)

Ideas begin as Discussions so the community can refine and prioritize them. Accepted work may later become an Issue.

Before participating, read [CONTRIBUTING.md](CONTRIBUTING.md), [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md), and [SECURITY.md](SECURITY.md).
